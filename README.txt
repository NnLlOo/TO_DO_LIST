Консольный менеджер задач (TODO List) на Python.

Этот проект представляет собой простое консольное приложение для управления списком задач, написанное на языке Python. Программа позволяет добавлять новые задачи, просматривать список всех задач, отмечать задачи как выполненные, удалять задачи, а также автоматически сохранять все данные в файл tasks.json. Благодаря этому список задач сохраняется между запусками программы и не теряется после закрытия приложения. Программа открывается в консольном приложении.

Для запуска необходимо установить Python, скачать файл main.py, открыть терминал в папке с проектом и выполнить команду: python main.py. После первого запуска в папке проекта автоматически появится файл tasks.json, в котором будут храниться все созданные задачи.

Каждая задача сохраняется в виде структуры данных с названием задачи и отметкой о выполнении. Проект можно расширять, добавляя дату выполнения, приоритеты, поиск и фильтрацию задач, цветной вывод в консоль и т.д.
